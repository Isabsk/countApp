# Count App

A premium, glassmorphic counter application built with Next.js.

## Features

- **Beautiful UI**: Modern glassmorphism design with animated background.
- **Interactive**: Smooth counters and controls.
- **Responsive**: Works on all devices.
- **Dark Mode**: Visual style optimized for dark environments.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- React
- Vanilla CSS (Glassmorphism)
